**Saving a task as a draft if helpful when:**

- You want time to plan complex or detailed prompts over multiple sessions
- You want to quickly jot down an idea for a task without having all of the needed context
- You want want to prepare multiple related tasks at the same time

### Saving Tasks as Drafts

<Steps>
  <Step>
    Start writing a prompt as usual from [<House className="inline-block w-4 h-4 mx-1" /> **Home**](https://www.terragonlabs.com)
  </Step>
  <Step>Click the dropdown arrow next to the submit button</Step>
  <Step>Select `Save as Draft`</Step>
  <Step>
    Your draft appears in the task list with a{" "}
    <File className="inline-block w-4 h-4 mx-1" /> draft indicator
  </Step>
</Steps>

### Submitting a Draft

When you're ready to run a draft task:

<Steps>
  <Step>Click on the draft task from your task list</Step>
  <Step>Review or edit the prompt if needed</Step>
  <Step>Click the submit button to run the task</Step>
</Steps>
