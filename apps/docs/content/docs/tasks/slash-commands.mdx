## Overview

Get access to built-in and custom slash commands (where supported by the agent) while writing prompts in Terragon. Type `/` in the prompt box to see available commands.

### Built-in Slash Commands

These slash commands work with all agents.

| Command    | Description                |
| ---------- | -------------------------- |
| `/clear`   | Clear conversation history |
| `/compact` | Compact conversation       |

### Custom Slash commands

You may also use other slash commands supported by the agent you are using. The following agents support custom slash commands:

- [Claude Code](/docs/agent-providers/claude-code)

See the agent provider documentation for more information on supported slash commands.

### Using Slash Commands

| Action                     | Shortcut           |
| -------------------------- | ------------------ |
| Trigger slash command menu | `/`                |
| Navigate menu              | `↑` `↓` Arrow keys |
| Select command             | `Enter` or `Tab`   |
| Cancel menu                | `Esc`              |
