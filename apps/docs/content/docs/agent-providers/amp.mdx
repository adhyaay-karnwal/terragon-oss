## Setup

To use Amp, you must connect your Amp API key. You can get your Amp API key from your [Amp Code Settings Page](https://ampcode.com/settings).

Configure your Amp API key in the [settings](https://www.terragonlabs.com/settings/agent) page.

## Features

### Slash Commands

Amp supports built-in slash commands:

- `/clear` - Clear conversation history
- `/compact` - Compact conversation

Custom slash commands (like `/init`, `/pr-comments`, `/review`) are not available with Amp. Learn more in the [Slash Commands documentation](/docs/tasks/slash-commands).
