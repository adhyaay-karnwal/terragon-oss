## Setup

There's no setup required for OpenCode. It works out of the box with our built-in credits system.

We pass the cost of these models transparently to you with zero markup or extra fees. You can find the pricing for the models in the links below.

## Available Models

OpenCode works with almost any model you can think of. For now, we've chosen a few models that perform well and are extremely cost-effective.

- [GLM 4.6](https://openrouter.ai/z-ai/glm-4.6:exacto)
- [Kimi K2](https://openrouter.ai/moonshotai/kimi-k2-0905:exacto)
- [Grok Code Fast 1](https://openrouter.ai/x-ai/grok-code-fast-1)
- [Qwen3 Coder 480B](https://openrouter.ai/qwen/qwen3-coder:exacto)
- [Gemini 2.5 Pro](https://openrouter.ai/google/gemini-2.5-pro)
